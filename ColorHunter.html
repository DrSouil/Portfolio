ColorHunter

ColorHunter_VR from hengshi kang

本作品是一个有关对颜色、形状的认识和分辨的手机VR游戏。使用Unity3D +Google Cardboard SDK 进行开发，运行在Android手机上。游戏场景为10m*10m*5m的虚拟空间，游戏中场景会生成很多各种颜色的立方体、球体和圆柱体，数量随得分而增加，游戏目标为寻找特定的某一物体。使用手机的陀螺仪和按键（也是Cardboard的按键）作为交互输入。以虚拟视选点作为交互手段和虚拟的VR场景进行交互。

This is a mobile phone VR game about color and shape recognition and identification. I used Unity3D +Google Cardboard SDK to developing. It runs on the Android mobile phone. The game scene is a 10m*10m*5m virtual space, where there will generate many different colors of cubes, spheres, and cylinders. The number increases with the score. The game goal is to find a particular object. It use the phone's gyroscope and button (or Cardboard button) as interactive input and the virtual view point to interact with the virtual scene.
